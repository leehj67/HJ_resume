# HJ_resume - 현실을 넘어선, 가상공간을 향하여

노션에서 내보낸 이력서/포트폴리오를 GitHub Pages로 게시하는 프로젝트입니다.

## 🌐 배포 URL

**https://leehj67.github.io/HJ_resume/**

## GitHub Pages 설정 (최초 1회)

1. [Settings > Pages](https://github.com/leehj67/HJ_resume/settings/pages) 이동
2. **Build and deployment** > **Source**에서 **GitHub Actions** 선택
3. 저장 후 1~2분 대기

## 로컬에서 미리보기

브라우저에서 `index.html` 파일을 직접 열어 확인할 수 있습니다.
