기술 스택 (Tech Stack)

  • OS/런타임: Windows, Python 3.13
  • 음성 합성: Windows SAPI SpVoice (보이스: Microsoft Heami / Microsoft Zira)
  • 오디오/영상 처리: FFmpeg (adelay, amix, sidechaincompress, scale/crop, subtitles)
  • 파이썬 스크립트: 표준 라이브러리 + pywin32 (TTS), numpy(로컬 징글)
  • 영상 소스: HeyGen 캐릭터(이미지→비디오), Custom Motion(손 흔들며 미소)
  • 서체(자막): 시스템 폰트 패밀리명 사용 (Malgun Gothic 등)

사용한 기능 (Features)

  • ✅ 로컬 징글 생성: 간단한 신디(벨/글로켄 느낌) 알고리즘으로 10–12초 배경음 자동 생성
  • ✅ SRT 자동 생성: TXT 가사를 10초 타임라인에 맞춰 SRT 변환
  • ✅ 보컬 합성(로컬 TTS): Windows SAPI(Heami/Zira)로 줄 단위 보컬 생성
  • ✅ 보컬 전처리/믹싱: 시간 정렬(adelay), 볼륨/정규화, (옵션) BGM 덕킹 or 제거
  • ✅ 세로 9:16 리프레임 + 자막 번인: FFmpeg 필터로 리스케일/크롭/자막 폰트 스타일
  • ✅ 자동 매칭 빌드: 파일명이 맞으면 메타 없이 자동 합성, 다르면 meta.jsonl로 명시

트러블슈팅 & 우리가 해결한 이슈

  • pydub/audioop 에러 (Py 3.13) → audioop 제거로 인해 발생 → pydub 미사용 경량 경로로 변경
  • 자막 폰트 에러 → FontName에 패밀리명 사용 + \, 이스케이프
  • 보컬 묻힘 → 보컬 volume=+12dB, BGM volume=-8~-18dB, sidechaincompress 강화
  • Here-doc/줄바꿈 오류(PowerShell) → 백틱 사용/한 줄 실행 가이드 제공
  • rubberband pitch 에러 → semitone→ratio 변환(2^(semi/12)), 입력 인덱스 정정
