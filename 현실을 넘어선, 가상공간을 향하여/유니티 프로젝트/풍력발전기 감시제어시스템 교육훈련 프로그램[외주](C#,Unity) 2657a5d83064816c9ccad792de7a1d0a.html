프로젝트 소개

2021년 6월부터 12월까지, Unity를 활용하여 두 개의 주요 프로젝트에 참여했습니다. 첫 번째 프로젝트인 [어린아이 눈 운동 VR 어플] 개발에서의 성공적인 성과 덕분에, 두 번째 프로젝트인 [풍력발전 감시제어 시스템 교육훈련 프로그램] 개발에 참여할 수 있었습니다.

두 번째 프로젝트에서는 학습자들이 프로그램의 복잡한 기능을 직관적으로 이해하고, 효율적으로 학습할 수 있도록 사용자 맞춤형 UI를 설계했습니다. 또한, 풍력 발전기 동작과 터빈 분리 등 다양한 고급 애니메이션을 제작하여 학습자의 이해를 돕고 몰입감을 높였습니다. 프로그램의 초반 단계에서 학습자가 쉽게 접근할 수 있도록 튜토리얼과 가이드 애니메이션을 개발하였고, 음성 자막(TTS) 시스템을 통합하여 다양한 학습 스타일을 지원했습니다.

  • 인트로 및 사용자 설정 UI 제작: 사용자 친화적인 인터페이스를 설계하여 학습자가 프로그램을 직관적으로 사용할 수 있도록 도왔습니다.
  • 애니메이션 설계 및 구현: 풍력 발전기 동작, 터빈 분리 등 다양한 애니메이션을 제작하여 학습자의 이해를 돕고 몰입감을 높였습니다.
  • 튜토리얼 및 가이드 애니메이션 개발: 프로그램 초기 단계에서 학습자가 기능을 쉽게 이해할 수 있도록 인터랙티브 가이드 애니메이션을 도입하여 학습 효율성을 크게 향상시켰습니다.
  • TTS 음성 및 자막 시스템 통합: 자막과 음성 안내 시스템을 구축하여 다양한 학습 스타일을 지원하고, 학습자의 편의성을 높였습니다.
  • 씬 전환 및 데이터 관리: 이전 화면으로 되돌아가는 기능을 포함해, 프로그램 전반의 씬 전환과 데이터 관리를 최적화하여, 일관되고 자연스러운 사용자 경험을 제공했습니다.
    이와 같은 기여를 통해, 프로젝트의 교육적 효과와 사용자 경험을 크게 개선하며, 기업의 핵심 Unity 프로젝트에 중요한 역할을 수행했습니다.

문제사항 및 해결방법

문제사항
  • 학습자들이 프로그램의 복잡한 기능을 직관적으로 이해하고, 효율적으로 학습할 수 있도록 돕는 것이 필요했습니다.
  • 학습자가 이전 화면으로 되돌아갈 때 애니메이션이 부자연스럽게 재생되는 문제가 발생했습니다.
  • 학습자가 UI 내에서 특정 학습 위치로 이동한 후, 초기 설정으로 자동 초기화되지 않는 문제가 있었습니다.
해결방법
  • 인터랙티브한 가이드 애니메이션을 추가하여, 프로그램의 시작 단계에서 각 버튼과 기능의 목적과 사용 방법을 명확히 설명하였습니다. 이를 통해 학습자들이 프로그램의 흐름을 쉽게 이해하고 따라갈 수 있게 했습니다.
  • 재생과 역재생 애니메이션을 별도로 설계하고, 각 씬에 넘버링 시스템을 도입하여, 사용자가 이전 화면으로 돌아갈 때 자동으로 역재생 애니메이션이 동작하도록 구성했습니다. 이로 인해 학습 경험이 더 자연스러워졌습니다.
  • 씬 전환 시 데이터를 유실하지 않도록 DontDestroyOnLoad를 사용하여 현재 위치와 상태 데이터를 유지하도록 했으며, 씬 이동이 감지될 때마다 씬 데이터가 자동으로 초기화되도록 설정하여 일관된 사용자 경험을 제공했습니다.
